Menghitung Median (Nilai Tengah)
Definisi Median
Median adalah nilai tengah dari data yang sudah diurutkan.

Untuk n genap: Median = (data ke-n/2 + data ke-(n/2+1)) / 2
Data Terurut

Nilai160162163164165166167168169190
Perhitungan
n = 10 (genap) Data ke-5 = 165 Data ke-6 = 166 Median = (165 + 166) / 2 = 331 / 2 = 165.5 cm
Median tinggi badan = 165.5 cm
